### 👾 Instalaciónes BOT 👾

<details>
<summary><strong>⚡ Cloud</strong> — Shell</summary>

```bash
git clone damianmoran2025-sketch/MICET-BOT-MD-ULTRA-BOT
```

```bash
cd MICET-BOT-MD-ULTRA-BOT
```

```bash
yarn install
```

```bash
npm install
```

```bash
npm start
```

</details>
